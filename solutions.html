Our Accessibility Solutions

Braille Guidebook

Comprehensive botanical art guide featuring braille descriptions of plants including Monstera, herbs, sunflower, and Japanese Maple. Provides tactile learning experience for visually impaired visitors.

Enhanced Signage

Upgraded existing plant identification signs with braille text. Example shows Corsican Pine (Pinus nigra ssp. laricio) with QR code integration for additional digital accessibility.

Digital Platform

Redesigned website interface following accessibility standards, featuring clear navigation, botanical information, and integration with physical braille solutions.

Detailed Solution Features

Braille Integration System

  • Grade 2 Braille: Professional-quality braille text for experienced users
  • Weather Resistance: Specialized materials designed for outdoor Scottish conditions
  • Tactile Graphics: Raised illustrations showing plant structures and characteristics
  • Modular Design: Easy to update and expand with new plant information

Digital Accessibility Features

  • QR Code Integration: Instant access to audio descriptions via smartphone
  • Screen Reader Compatibility: Full NVDA and JAWS support
  • Voice Navigation: Hands-free browsing and information access
  • Offline Capability: Downloaded content for areas with poor connectivity

Enhanced Signage System

  • Dual Format: Visual and braille information on every sign
  • Consistent Placement: Standardized height and positioning for predictability
  • High Contrast Design: Improved visibility for low vision users
  • Multi-language Support: English, Gaelic, and international accessibility

Implementation Strategy

Phase 1: Pilot Installation

Install braille signage and guidebooks in the main greenhouse area for initial testing and feedback collection.

Phase 2: Digital Integration

Launch QR code system and accessible website platform with full audio description library.

Phase 3: Garden-wide Rollout

Expand accessibility features to all major garden areas and visitor pathways.

Phase 4: Evaluation & Refinement

Comprehensive user testing and system optimization based on visitor feedback and usage data.
